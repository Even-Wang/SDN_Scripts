## NIC_rate_recorder_monitor.sh

monitor one NIC's Rx/Tx rate, and record them in .csv file

For example, run: 

    ./NIC_rate_recorder_monitor.sh eth1

Data file format:

    Moment(seconds),RX(Mbps),TX(Mbps)
